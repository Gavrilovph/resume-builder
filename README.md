# course_work_progress
Nice and easy cv builder. Vue + Reactive components + Axios for comments request.
To add an avatar please use a url of a pic.

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
